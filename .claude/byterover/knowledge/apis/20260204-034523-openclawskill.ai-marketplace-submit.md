- OpenClaw Skill (openclawskill.ai) is the official registry and marketplace for AI skills compatible with OpenClaw, ClawBot, and MoltBot platforms.
- Submit skills by creating custom SKILL.md files with natural language instructions - no fine-tuning required.
- The marketplace is open source, community-driven with zero barriers to entry.
- Skills are installed using command-line tools like `npx openclawskill install [skill-name]`.
- The platform offers versioning, rollback functionality, and discoverable skills through intelligent search.
